🌿 Kishan Copilot
AI-Powered Smart Farming Assistant for Indian Farmers
📌 Overview
Kishan Copilot is an AI-driven agriculture assistance platform designed to empower Indian farmers with intelligent, real-time, and accessible farming support.

The platform combines modern AI technologies with practical agricultural workflows to help farmers make better decisions related to:

Crop selection
Disease diagnosis
Weather-based farming decisions
Market price tracking
Government scheme awareness
Multilingual AI assistance
Built with a focus on accessibility, simplicity, and real-world utility, Kishan Copilot aims to bridge the gap between advanced agricultural intelligence and grassroots farming communities.

🚀 Features
🌱 Intelligent Crop Advisory
Farmers can enter:

Soil type
Location
Season
Farm size
Previous crop history
The AI system then recommends:

Best crops to grow
Fertilizer recommendations
Irrigation schedules
Yield estimates
Sowing timelines
🔬 AI Disease Diagnosis
Farmers can upload crop images and receive:

Disease detection
Pest identification
Nutrient deficiency analysis
Treatment recommendations
Preventive measures
This helps reduce crop loss and enables early intervention.

☁️ Hyper-Local Weather Insights
The platform provides:

Real-time weather conditions
Temperature
Humidity
Wind speed
Farming-specific weather advice
Helping farmers decide:

Irrigation timing
Spraying schedules
Harvest timing
📈 Market Price Intelligence
Farmers can check mandi and crop prices including:

Minimum price
Maximum price
Modal price
Market trends
Best mandis for selling
This helps maximize profits and improve selling decisions.

🏛️ Government Scheme Discovery
The system helps farmers discover important schemes such as:

PM-KISAN
Kisan Credit Card
Crop Insurance
Subsidy Programs
Including:

Eligibility
Benefits
Application process
Relevant links
🌐 Multilingual Support
Kishan Copilot supports multilingual interaction for wider accessibility among Indian farmers.

Current support:

⚙️ Installation & Setup
1. Clone Repository
git clone <your-repository-link>
cd kishan-copilot
2. Install Dependencies
pip install flask requests python-dotenv groq
3. Create .env File
Create a .env file in the root directory.

GROQ_API_KEY=your_groq_api_key
WEATHER_API_KEY=your_weather_api_key
PLANT_API_KEY=your_plant_api_key
4. Run the Application
python app.py
5. Open in Browser
http://127.0.0.1:5000
